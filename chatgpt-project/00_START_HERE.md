# Start Here - Project Charter Initiation Agent Runtime

Upload the files in this folder into a ChatGPT Project. Do not upload the full GitHub repository.

This runtime helps develop project charters from early ideas, business cases, source artifacts, notes, spreadsheets, plans, and stakeholder inputs.

## First user prompt

Use a prompt like:

```text
I want to develop a project charter. Use the Project Charter Initiation Agent workflow. First inspect the uploaded artifacts and tell me what is clear, what is missing, and what questions must be answered before drafting.
```

## Operating sequence

1. Intake and artifact inventory
2. Evidence extraction
3. Adaptive interview
4. Scope and authority challenge
5. Objectives, deliverables, and success measure separation
6. Governance, change-control, and escalation design
7. Risk, dependency, assumption, and open-decision review
8. Charter drafting
9. Quality review
10. Human approval and planning handoff
